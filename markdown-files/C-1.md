# C-#1

## 认识C
1.诞生:</br>
C诞生于`阿尔卡特朗讯贝尔实验室`(Alcatel-Lucent Bell Labs),由丹尼斯·麦卡利斯泰尔·里奇(Dennis MacAlistair Ritchie,DMR,1941.9.9 ~ 2011.10.12,已逝世)创建.C以B为基础.C的扩展名为`.c`.</br>

2.用C写成的伟大作品:</br>
`Unix`(丹尼斯·麦卡利斯泰尔·里奇,Dennis MacAlistair Ritchie,DMR)</br>
`Linux`(林纳斯·本纳第克特·托瓦兹,Linus Benedict Torvalds,LBT)</br>

## 搭建C的环境
选项1:使用文本编辑器+C编译器</br>

文本编辑器:</br>
`记事本`(微软,Windows)</br>
`Notepad++`(Don HO,Windows)</br>
`GNU Emacs`(GNU计划,Linux/Windows/MacOS)</br>
`Vim`</br>

C编译器:</br>
Linux:</br>
`gcc*`(自带)</br>
Windows:</br>
`Mingw`(gcc的Windows移植版,只能生成32位程序)</br>
`Mingw64*`(gcc的Windows移植版,能生成64位程序)</br>
`cl`(微软Visual Studio自带)</br>
`LLVM+Clang`</br>
MacOS:</br>
`LLVM+Clang`</br>
`gcc*`</br>

附言:上述标记了`*`为推荐使用的编译器.Windows在安装`Mingw64`时需要在环境变量`Path`中添加`Mingw64`的路径中的`bin`目录的绝对路径.</br>

选项2:使用集成开发环境(Integrated Development Environment,IDE)</br>

`微软Visual Studio`(VS,Windows,社区版免费)</br>
`Eclipse`(Linux/Windows/MacOS,开源,免费)</br>
`Xcode`(MacOS)</br>
`微软Visual Studio Code`(VSC,Linux/Windows/MacOS,轻量级,开源,免费)</br>
`Atom`(Linux/Windows/MacOS,轻量级,开源,免费)</br>
`CLion`(Linux/Windows/MacOS,收费)</br>

附言:部分集成开发环境需要额外配置gcc.完成配置gcc后,请在`命令提示符`(Command Prompt,CMD),`PowerShell`或`终端`(Terminal)中输入`gcc -v`来检查是否安装成功</br>

## 你好世界
1.打开你的`文本编辑器`或`集成开发环境`.</br>
2.添加以下代码.</br>
``` c
#include <stdio.h>
int main {
    printf("你好世界");
    return 0;
}
```

选项1:使用文本编辑器+C编译器</br>
3.保存为`helloworld.c`.</br>
4.在文件的目录下使用打开`命令提示符`,`PowerShell`或`终端`.</br>
5.输入`gcc helloworld.c`</br>
6.输入`a.exe`(Windows),`a.out`(Linux)或`a`(MacOS)</br>

附言:使用`gcc`时生成文件可能不同,视情况而定.</br>

选项2:使用集成开发环境</br>
6.点击启动(快捷键一般为<kbd>F2</kbd>)</br>

输出结果:</br>
`你好世界`</br>
如果你到这一步,你成功地制作了你的第一个C程序.
