## 知识解析 ##
### 长度 ###
1.长度的单位  
在国际单位制中,长度的单位是米(metre(英)/minter(美),符号:"m")  
常用单位:千米(km),分米(dm),厘米(cm),毫米(mm),微米(μm/(可写)um),纳米(nm)  
  
2.长度单位的换算关系  
1 km = 10<sup>3</sup> m;1 m = 10<sup>3</sup> mm = 10<sup>6</sup> μm = 10<sup>9</sup> nm  
  
拓展  
1.m:光在真空中(1/299 792 458)s时间间隔内所经过路径的长度.**[第17届国际计量大会(1983)]**  
2.长度(length):一维空间的度量,为点到点的距离  
  
3.长度测量工具  
常用的长度测量工具:刻度尺  
高精度的长度测量工具:游标卡尺,螺旋测微器  

4.长度测量步骤  
1)认  
认清刻度尺上所标的单位,零刻度线(刻度尺的起始刻度)的位置,量程(刻度尺的测量范围)和分度值(相邻两刻度线之间的距离)  
![认识刻度尺](image-files/zk-physical-review-outline(fj)/认识刻度尺.png "认识刻度尺")  
2)选  
在事先没有给定刻度尺时,要根据测量要求选择适当的刻度尺.选择时要"两选":  
(1)量程:量程要合适,尽量一次就测完被测物体的全长  
(2)分度值:根据需要,分度值要合适,不能太大,也不能太小  
3)放  
零刻度线或某一刻度线对准被测物体的一端,有刻度线的一边要紧靠被测物体且与被测物体对齐,不能歪斜  
![尺子的使用(错误示范)](image-files/zk-physical-review-outline(fj)/尺子的使用(错误示范).png "尺子的使用(错误示范)")  
![尺子的使用(正确示范)](image-files/zk-physical-review-outline(fj)/尺子的使用(正确示范).png "尺子的使用(正确示范)")  
4)看  
读数时,视线正对刻度线  
![读数](image-files/zk-physical-review-outline(fj)/读数.png "读数")  
5)读  
测量时要估读到分度值的下一位  
6)记  
记录测量结果时,要写出数字和单位  
### 时间 ###
1.时间的单位  
在国际单位制中,时间的基本单位是秒(second,符号"s")  
常用单位:时(hour,s),分(minute,m),毫秒(ms),微秒(μs/(可写)us)  
  
2.时间单位的换算  
1 h = 60 min,1 min = 60 s,1 s = 10<sup>3</sup> ms,1 ms = 10<sup>3</sup> μs  
  
拓展  
1.s:铯-133原子基态的两个超精细能级之间跃迁所对应的辐射的9 192 631 770个周期的持续时间.**[第13届国际计量大会(1967),决议1]**  
2.时间(time):物质的运动,变化的持续性,顺序性的表现/人类用以描述物质运动过程或事件发生过程的一个参数  
  
3.时间测量工具  
常用的长度测量工具:停表  
高精度的长度测量工具:石英钟  
### 误差 ###
1.定义:测量值与真实值之间的差别,叫做误差  
2.与错误的区别  
  
|  | 误差 | 错误 |
| - | :-: | :-: |
| 产生原因 | (1)跟测量工具的精确度有关<br/>(2)跟估读人读取估读值不同有关<br/>(3)跟测量方法有关 | (1)不遵守测量仪器的使用规则<br/>(2)读取,记录结果时粗心 |
| 减小或避免 | 不可避免,只可减少 | 可以避免 |
  
3.减少误差的方法  
1)多次测量求平均值  
2)选用精密仪器  
3)改进测量方法  
### 机械运动 ###
1.在物理学中,物体随时间的变化叫做机械运动  
2.参照物:用于判断物体是否运动所选取作为标准的物体,叫做参照物  
如果一个物体的位置相对于参照物发生了变化,他是运动的;如果一个物体的位置相对于参照物没有发生了变化,他是静止的  
3.选择参照物的注意事项  
1)任意性:参照物的选择可以是任意的,既可以是运动的物体,亦可以静止的物体  
2)排己性:参照物不能选择研究对象本身.因为若以自己为参照物,研究对象的位置是不可能发生变化的  
3)不唯一性:参照物的选择不唯一,可以选择不同的参照物,若选择了不同的参照物,研究对象的运动状态的描述往往是不同的  
4)方便性:通常情况下,为了研究方便,一般选取地面或相对于地面静止的物体作为参照物,且可以不加说明;若选取其他合适的物体作为参照物来研究机械运动时,则要做出说明  
5)假定性:参照物一旦选择,就假定参照物是"静止"的  
  
3.运动与静止  
对于一个物体,选择的参照物不同,运动的情况可能不同  
同一个物体对于另一个物体可能是静止的,也有可能是运动的,这就是运动的相对性
### 速度 ###
1.比较物体快慢的方法  
1)通过相同的时间比较路程  
2)经过相同的路程比较时间  
  
2.速度  
1)定义:在物理学中,把路程和时间之比叫做速度  
2)意义:表示物体的快慢
3)符号: "v"  
4)公式:v = s / t,s = vt,t = s / v  

3.速度的单位  
米/秒(符号:m/s),千米/时(符号:km/h)  

4.速度单位的换算    
1 m/s = 3.6 km/h  
### 运动类型 ###
1.匀速直线运动  
1)概念:物体运动速度不变的直线运动  
2)特点:运动过程中,沿直线运动,速度大小和运动方向不改变  
  
2.变速直线运动  
1)概念:物体运动速度变化的直线运动  
2)特点:运动过程中,沿直线运动,速度大小改变,运动方向不改变  










# 物理相关 #
1.国际单位制(Système International d'Unités(法语),简称:"SI"):由 "国际计量大会" 制定的一套标准  

2.SI基本单位  
  
| 量的名称 | 单位名称 | 单位符号 |
| :-: | :-: | :-: |
| 长度 | 米 | m |
| 质量 | 千克 | kg |
| 时间 | 秒 | s |
| 电流 | 安培 | A |
| 热力学温度 | 开尔文 | K |
| 物质的量 | 摩尔 | mol |
| 发光强度 | 坎德拉 | cd |
  
3.SI基本单位的定义  
1)米:光在真空中(1/299 792 458)s时间间隔内所经过路径的长度.**[第17届国际计量大会(1983)]**  
2)千克:国际千克原器的质量.**[第1届国际计量大会(1889)和第3届国际计量大会(1901)]**  
3)秒:铯-133原子基态的两个超精细能级之间跃迁所对应的辐射的9 192 631 770个周期的持续时间.**[第13届国际计量大会(1967),决议1]**  
4)安培:在真空中,截面积可忽略的两根相距1 m的无限长平行圆直导线内通以等量恒定电流时,若导线间相互作用力在每米长度上为2×10-7 N,则每根导线中的电流为1 A.**[国际计量委员会(1946)决议2.第9届国际计量大会(1948)批准]**  
5)开尔文:水三相点热力学温度的1/273.16.**[第13届国际计量大会(1967),决议4]**  
6)摩尔:是一系统的物质的量,该系统中所包含的基本单元(原子、分子、离子、电子及其他粒子,或这些粒子的特定组合)数与0.012 kg碳-12的原子数目相等.**[第14届国际计量大会(1971),决议3]**  
7)坎德拉:是一光源在给定方向上的发光强度,该光源发出频率为540×1012 Hz的单色辐射,且在此方向上的辐射强度为(1/683)W/sr.**[第16届国际计量大会(1979),决议3]**
